
# 리뷰크롤링전처리팀
- 날짜 : 2021.03.11
- 진행사항 : 다나와, 올리브영, 네이버, 이니스프리 형태소 처리 완료

```
- 전처리 내용
  컬럼명과 순서: review, point, date, ingre, token_review
    - review : 리뷰
    - point : 모든 문자열 제거, 숫자만 남도록 처리
    - date : 불필요한 칼럼이라 판단 되어 삭제
    - ingre : 성분추가(하일루론산, 프로폴리스 등등)
    - token_reivew : konlpy 를 통한 형태소 분석
```
